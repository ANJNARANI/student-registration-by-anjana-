# Student Registration System

#File Link- 

A simple student registration system built with HTML, CSS and JavaScript.
Features:
- Add / Edit / Delete students
- Validation for name, student id, email and contact
- Client-side persistence using localStorage
- Responsive layout and simple accessible table

## Files
- `index.html` — markup (main page)
- `styles.css` — styling and responsive rules
- `app.js` — front-end logic, validation, localStorage

## How to run locally
Open `index.html` in any browser.
